# Random Password Generator

### Step 1

Clone the repository

```shell
git clone https://github.com/natashagp/random-password-generator.git
```

### Step 2

Change the directory to random-password-generator

```shell
cd random-password-generator
```

### Step 3

Run project

```shell
python main.py
```
